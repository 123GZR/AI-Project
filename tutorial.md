# Windows系统操作教程

## 系统信息查看

### 查看系统信息的方法
1. **使用系统信息工具**：
   - 按下Win + R键打开运行对话框
   - 输入`msinfo32`并按Enter
   - 系统信息窗口会显示详细的系统配置信息

2. **使用命令提示符**：
   - 按下Win + X，选择"Windows PowerShell"
   - 输入`systeminfo`命令
   - 可以看到完整的系统配置信息

3. **查看基本系统信息**：
   - 右键点击"此电脑"，选择"属性"
   - 在系统窗口中可以看到Windows版本、处理器和内存信息

## 文件和文件夹操作

### 创建文件夹
1. **图形界面方式**：
   - 在要创建文件夹的位置右键点击空白处
   - 选择"新建" > "文件夹"
   - 输入文件夹名称并按Enter

2. **命令行方式**：
   - 打开命令提示符或PowerShell
   - 使用`mkdir 文件夹路径`或`md 文件夹路径`命令
   - 例如：`mkdir D:\新建文件夹`

### 文件复制和移动
1. **复制文件**：
   - 选中要复制的文件，按Ctrl + C
   - 导航到目标位置，按Ctrl + V
   
2. **移动文件**：
   - 选中要移动的文件，按Ctrl + X
   - 导航到目标位置，按Ctrl + V
   - 或者直接拖拽文件到目标位置

## 常见Windows工具

### 磁盘清理
- 按下Win + R，输入`cleanmgr`并按Enter
- 选择要清理的驱动器
- 选择要删除的文件类型
- 点击"确定"开始清理

### 任务管理器
- 按下Ctrl + Shift + Esc直接打开任务管理器
- 或者按下Ctrl + Alt + Del，然后选择"任务管理器"
- 可以查看和结束运行中的进程，监控系统性能

### 控制面板
- 按下Win + R，输入`control`并按Enter
- 可以访问各种系统设置和配置选项

## 网络相关操作

### 查看网络连接
1. **使用网络和共享中心**：
   - 右键点击任务栏中的网络图标
   - 选择"打开网络和共享中心"
   - 可以查看当前网络连接状态

2. **命令行方式**：
   - 打开命令提示符
   - 输入`ipconfig`命令查看网络配置
   - 输入`ping 网址`测试网络连接

### 连接到无线网络
- 点击任务栏中的网络图标
- 选择要连接的无线网络
- 输入密码（如果需要）
- 点击"连接"

## 故障排除

### 常见问题解决方法

1. **程序无响应**：
   - 按下Ctrl + Shift + Esc打开任务管理器
   - 在"进程"选项卡中找到无响应的程序
   - 右键点击并选择"结束任务"

2. **系统运行缓慢**：
   - 使用磁盘清理工具清理临时文件
   - 检查任务管理器中占用资源较高的进程
   - 考虑增加内存或清理启动项

3. **无法连接到网络**：
   - 重启路由器和调制解调器
   - 检查网络适配器驱动程序
   - 尝试禁用并重新启用网络适配器

## Windows快捷键

### 常用快捷键
- **Win + D**: 显示桌面
- **Win + E**: 打开文件资源管理器
- **Win + L**: 锁定电脑
- **Ctrl + Shift + N**: 新建文件夹
- **Alt + F4**: 关闭当前窗口
- **Ctrl + Z**: 撤销
- **Ctrl + Y**: 重做
- **Ctrl + F**: 查找

### 高级快捷键
- **Win + X**: 打开快速访问菜单
- **Win + P**: 投影设置
- **Win + I**: 设置
- **Shift + Delete**: 永久删除文件

## 系统维护

### 定期维护任务
1. **Windows更新**：
   - 打开设置 > 更新和安全
   - 点击"检查更新"
   - 安装可用的更新

2. **病毒扫描**：
   - 确保Windows Defender或其他安全软件已更新
   - 定期执行完整系统扫描

3. **备份数据**：
   - 使用Windows备份工具或第三方备份软件
   - 定期备份重要文件和数据
